# AI Interview Coach

A complete production-style application utilizing the OpenAI API to evaluate candidates' interview readiness.

## Features
- Phase 1: Setup & Initialization
- Phase 2: OpenAI Question Generation (5 Technical, 2 Behavioral)
- Phase 3: AI Answer Evaluation with scoring and actionable feedback
- Phase 4: UI interactive end-to-end flow
- Phase 5: Polish & UX enhancements

## Run

```bash
export OPENAI_API_KEY="your-api-key"
cd src
flask run
```
